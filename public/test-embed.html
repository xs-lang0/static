XS Embed SDK - Test Page

Testing all three embedding modes plus auto-enhance.

1. <xs-editor> - Default

Interactive editor with run button. Click "Run" or press Ctrl+Enter.

println("hello from XS!") for i in 1..5 { println("count: {i}") }

2. <xs-editor> - Custom styled

Same component, different theme via CSS variables.

fn describe(val) { match val { 0 => "zero" n if n > 0 => "positive: {n}" _ => "negative" } } println(describe(42)) println(describe(0)) println(describe(-7))

3. <xs-editor> - Run on load, readonly

let xs = "XS" println("{xs} runs in the browser!")

4. <xs-code> - Auto-executing

Shows the code and runs it automatically.

fn fib(n) { if n <= 1 { return n } fib(n - 1) + fib(n - 2) } for i in 0..10 { println("fib({i}) = {fib(i)}") }

6. Auto-enhanced code blocks (data-auto)

These are regular <pre><code class="language-xs"> blocks. Hover to see the "Run" button.

fn* count_up(start) {
  var n = start
  loop {
    yield n
    n = n + 1
  }
}

for n in count_up(1) {
  if n > 5 { break }
  println(n)
}
let msg = "hover over me and click Run!"
println(msg)
